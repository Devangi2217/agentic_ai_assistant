# Agentic AI Assistant (Tool Orchestration)

A mobile-first agentic AI assistant concept with interactive workflow steps, tool orchestration logs, validation loops, and DataVault storage controls. Built with Expo + React Native.

## Features
- Workflow step toggles (Pending → Running → Done)
- Tool execution mock logs
- Validation pass/fail simulation
- DataVault storage controls
- Clean, single-screen navigation for demo workflows

## Tech Stack
- React Native
- Expo
- JavaScript

## Live Demo (Web)

```bash
npm install
npx expo start
```
